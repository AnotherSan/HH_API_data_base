# HH_API_data_base
Проект создан в рамках проектного практикума для УРФУ. 
Это парсер данных с API HeadHunter, предназначенный для составления статистики по рынку труда.
### Запуск скрипта на Linux
* cd папка загрузки
* sudo nano api.py
### Запуск скрипта на Windows
* Win+R
* cmd
* cd папка загрузки
* python>api.py
